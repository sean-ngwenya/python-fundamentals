# Python Fundamentals

A structured and practical refresher of core Python concepts, organized by topic and reinforced with small, focused examples.

This repository is designed to consolidate Python fundamentals before progressing to advanced libraries, frameworks, and real-world projects.

---

## Objectives

- Rebuild strong foundations in Python syntax and semantics
- Practice clean, readable, and idiomatic Python
- Reinforce concepts with small, runnable examples
- Establish professional repository structure and Git hygiene

---
